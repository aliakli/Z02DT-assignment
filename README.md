# Z02DT-assignment
## How to use
- help for commands
- test for calculator test (must have administrator priviliges)

### ====== Trig functions ======
*Default Radians*
- sin(angle)
- cos(angle)
- tan(angle)

### ====== Inverse Trig Functions ======
*Default Radians*
- arcsin(ratio)
<br> must be 1>=ratio>=-1
- arccos(ratio)
<br> must be 1>=ratio>=-1
- arctan(ratio)

### ====== Radical Functions ======
*All results must be on the real plane*
- sqrt(arg)
<br> must not be negative
- cbrt(arg)
- n_rt(arg, root)
<br> if root is even, root cannot be negative

### ====== Logarithms ======
- log(arg)
<br>-> log base 10
- ln(arg)
<br>-> log base e
- log_n(arg, base)
<br>-> log base {base}

### ====== Constants ======
- pi = 3.141592653589793
- e = 2.718281828459045
